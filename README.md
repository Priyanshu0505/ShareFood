# рҹ“– About

Food wastage is a major global issue while millions of people struggle with hunger every day.

**ShareFood** is a full-stack web application that enables users to donate surplus food and allows NGOs or needy individuals to request available donations. The platform simplifies the donation process and helps reduce food waste through a modern, user-friendly interface.
