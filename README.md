# 📖 About

Food wastage is a major global issue while millions of people struggle with hunger every day.

**ShareFood** is a full-stack web application that enables users to donate surplus food and allows NGOs or needy individuals to request available donations. The platform simplifies the donation process and helps reduce food waste through a modern, user-friendly interface.

---

# ✨ Features

- 🍱 Donate surplus food
- 👤 User Authentication (Login & Signup)
- 📦 Manage food donations
- 📍 Location-based donation details
- ❤️ Request available food
- 📊 User Dashboard
- 🔐 Secure JWT Authentication
- 📱 Responsive Design
- ⚡ Fast and Modern UI
- ☁️ MongoDB Database Integration

# 🚀 Running the Project

## Frontend

```bash
npm install
npm run dev
```

## Backend

```bash
cd backend
npm install
npm start
```

---

# ⚙️ Environment Variables

Create a **.env** file inside the backend directory.

```env
PORT=5000

MONGODB_URI=Your MongoDB Connection String

JWT_SECRET=Your Secret Key
```
